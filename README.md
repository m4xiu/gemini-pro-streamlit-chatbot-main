#🤖 Gemini AI Chatbot (Streamlit)

An interactive AI chatbot built using Streamlit and Google Gemini API, enabling real-time conversations with a clean and user-friendly interface.

⸻

🚀 Features
	•	💬 Real-time AI chat powered by Gemini
	•	🧠 Session-based conversation memory
	•	🔐 Secure API key handling using .env
	•	🎨 Simple and clean Streamlit UI
	•	⚡ Fast and lightweight application
	•	🛠️ Easy to customize and extend

⸻

🛠️ Tech Stack
	•	Python
	•	Streamlit
	•	Google Generative AI (Gemini)
	•	python-dotenv

⚙️ Installation & Setup

1. Clone the repository
 git clone https://github.com/your-username/gemini-chatbot.git
cd gemini-chatbot

3. Create virtual environment (recommended)
   python -m venv venv
source venv/bin/activate   # Mac/Linux
venv\Scripts\activate      # Windows

4. Install dependencies
   pip install -r requirements.txt

🔐 Environment Variables

Create a .env file in the root directory:
    GOOGLE_API_KEY=your_api_key_here
    ⚠️ Never share your API key publicly

⸻

▶️ Run the App

  streamlit run main.py
🧠 How It Works
	•	Uses Google Gemini API for generating responses
	•	Maintains chat history using Streamlit session state
	•	Loads API key securely from .env
	•	Provides interactive chat UI using Streamlit components
